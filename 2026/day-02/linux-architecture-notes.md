<H2>Core components of Linux :</H2>
<br>Hardware : This is the physical component where hardware is.</br>
<br>Kernel : This is the heart of linux system. Linus korwald code written in C. It interacts with the hardware and take the shell commands</br>
<br>Shell : Shell is an interactive way to interact with kernel. Shell commands interact with Kernal i.e giving instructions to kernal to perform operations.</br>
<br>GUI : Graphical user interface for visual interaction.</br>
<br>System Libraries : System libraries are pre defined and user can use it to interact with system</br>
<br>System Utilities : It is basically shell commands like ps, ls that are given by user to perform operations like file management, System configration etc.</br>

<H2>Processes in Linux</H2>
<br>Everything in linux is process. commands like ps, top are used to display processes running on the system.</br>

<H2>Process states</H2>
<br>running :  Process are in Active mode</br>
<br>sleeping : System is in Idel mode</br>
<br>Stopped : Process Stopped</br>
<br>Zombie : The process is terminated but its parent entry is still exsited</br>

<br>Pwd: Present working directory where user knows where he is in current folder</br>
<br>ls: List files</br>
<br>df -h and free -h: This shows the free space on the disk and free ram available</br>
<br>ping <website>: This command is used to check wether the packages are sent and received properly</br>
