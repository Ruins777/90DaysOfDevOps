<H2>Core components of Linux :</H2>
<b></b></b>Hardware : This is the physical component where hardware is.</br>
<b>Kernel : This is the heart of linux system. Linus korwald code written in C. It interacts with the hardware and take the shell commands</b>
<b>Shell : Shell is an interactive way to interact with kernel. Shell commands interact with Kernal i.e giving instructions to kernal to perform operations.</b>
<b>GUI : Graphical user interface for visual interaction.</b>
<b>System Libraries : System libraries are pre defined and user can use it to interact with system</b>
<b>System Utilities : It is basically shell commands like ps, ls that are given by user to perform operations like file management, System configration etc.</b>

<H2>Processes in Linux</H2>
<b>Everything in linux is process. commands like ps, top are used to display processes running on the system.</b>

<H2>Process states</H2>
<b>running :  Process are in Active mode</b>
<b>sleeping : System is in Idel mode</b>
<b>Stopped : Process Stopped</b>
<b>Zombie : The process is terminated but its parent entry is still exsited</b>
