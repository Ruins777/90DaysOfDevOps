<H2>Core components of Linux :</H2>
Hardware : This is the physical component where hardware is. 
Kernel : This is the heart of linux system. Linus korwald code written in C. It interacts with the hardware and take the shell commands
Shell : Shell is an interactive way to interact with kernel. Shell commands interact with Kernal i.e giving instructions to kernal to perform operations.
GUI : Graphical user interface for visual interaction.
System Libraries : System libraries are pre defined and user can use it to interact with system
System Utilities : It is basically shell commands like ps, ls that are given by user to perform operations like file management, System configration etc.

<H2>Processes in Linux</H2>
Everything in linux is process. commands like ps, top are used to display processes running on the system.

<H2>Process states</H2>
running :  Process are in Active mode
sleeping : System is in Idel mode
Stopped : Process Stopped
Zombie : The process is terminated but its parent entry is still exsited
